# Udemy Clone Website

A simple frontend clone of the Udemy homepage built using HTML5 and CSS3.

## 🚀 Features

- Responsive-inspired homepage layout
- Modern navigation bar
- Search bar UI
- Course recommendation sections
- Topics section
- Popular courses section
- Footer section
- Flexbox-based layout design

## 🛠️ Technologies Used

- HTML5
- CSS3
- Font Awesome Icons
- Google Fonts

## 📚 Learning Outcome

This project helped me learn:
- Flexbox layout
- Website structuring
- Responsive design basics
- UI design concepts
- Frontend development fundamentals

## ⚠️ Note

This project currently focuses only on frontend UI design.  
Buttons, icons, and links are not functional yet.
